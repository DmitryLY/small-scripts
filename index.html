Поиск в поле SELECT

Исходный код целевого элемента select:

   data-value="0", data-label="Пермь" - атрибуты элемента select, которые указывает на опцию со значением value выбранной по умолчанию.

    <select name="" data-value="0" data-label="Пермь">    
         <option value="0">Пермь</option>
         <option value="1">Москва</option>
    </select>

Пример вызова функции:

   custom_select_generator({
      element: custom_select, // целевой элемент select
      change: set_option, // функция, которая будет вызвана при выборе опции из сгенерированного списка 
      placeholder: 'Выберите опцию', // текст, в placeholder видимого элемента input
      arrow: true // будет ли стрелка для открытия списка опций
   });

Сгенерированный код скриптом:

      Скрытое поле input с именем(name) элемента select и value, принимает value элемента из сгенерированного списка при его выборе.

    Видимое поле input примет значение самого тэга option в качестве атрибута value.
